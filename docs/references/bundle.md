# `bundle`: bundle & Watch

## Description


 Bundle and watch the JavaScript.


## Usage

```r
bundle(mode = c("production", "development", "none"))
bundle_prod()
bundle_dev()
watch()
```


## Arguments

Argument      |Description
------------- |----------------
```mode```     |     The configuration mode tells webpack to use its built-in optimisations accordingly.

## Examples

```r 
 watch() 
 
 ```
